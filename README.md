Cover letter 的模版

精投的时候，最好带上 cover letter，根据 JD 微调自己的简历。可以让 chatGPT 帮你写 Cover letter，但是必须自己删，因为 chatGPT 写的太啰嗦。

Cover letter 就写三段：第一段申请的什么职位，第二段经验怎么 match，第三段对这个职位很感兴趣，赶紧叫我面试
